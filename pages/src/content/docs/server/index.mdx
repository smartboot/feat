
## 建议顺序

如果你是第一次接触 Feat Core，建议这样读：

1. [构建第一个 Feat HTTP 服务](/feat/server/getstart/)
2. [Router 路由组件](/feat/server/router/)
3. [异步响应](/feat/server/async/)
4. [ServerOptions 配置指南](/feat/server/serveroptions/)

`HTTPS` 和 `协议升级` 更适合在你已经有实际需求时按需查阅。
