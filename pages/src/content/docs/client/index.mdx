Feat Core 客户端提供轻量级、高性能的 HTTP/SSE/WebSocket 客户端实现，适合需要从 Java 程序主动连接外部服务、处理流式响应或建立双向实时通信的场景。

## 客户端类型选择

| 客户端类型 | 适用场景 | 通信模式 |
|-----------|---------|---------|
| **HttpClient** | REST API 调用、文件下载 | 请求-响应 |
| **SSE Client** | 实时推送、日志流、监控数据 | 服务端单向推送 |
| **WebSocket Client** | 聊天室、协作编辑、游戏 | 双向实时通信 |

## 第一个 HTTP 请求

### 前置要求

- **JDK**: 8 或更高版本
- **构建工具**: Maven 3.0+ 或 Gradle
- **依赖**: `feat-core`

### 添加依赖

<Tabs>
<TabItem label="Maven">

```xml title="pom.xml"
<dependency>
    <groupId>tech.smartboot.feat</groupId>
    <artifactId>feat-core</artifactId>
    <version>${feat.version}</version>
</dependency>
```

</TabItem>
<TabItem label="Gradle">

```groovy title="build.gradle"
dependencies {
    implementation 'tech.smartboot.feat:feat-core:${feat.version}'
}
```

</TabItem>
</Tabs>

<Aside type="note">
`${feat.version}` 请替换为实际版本。最新版本可在 [Maven Central](https://central.sonatype.com/artifact/tech.smartboot.feat/feat-core) 查看。
</Aside>

### 编写代码

```java title="HttpGetDemo.java"
import tech.smartboot.feat.Feat;

public class HttpGetDemo {
    public static void main(String[] args) {
        Feat.httpClient("https://smartboot.tech")
                .get("/feat/")
                .onSuccess(response -> System.out.println(response.body()))
                .onFailure(Throwable::printStackTrace)
                .submit();
    }
}
```

### 核心概念

```
Feat.httpClient(baseUrl)  →  创建客户端
        ↓
    .get(path)            →  定义请求
        ↓
    .onSuccess(callback)   →  成功回调
    .onFailure(callback)   →  失败回调
        ↓
    .submit()              →  执行请求
```

<Aside type="tip">
`submit()` 返回 `CompletableFuture<HttpResponse>`，可与其它异步流程组合使用。
</Aside>

## 常见问题

### 连接失败："localhost:8080" 报错

**原因**: `HttpClient(String url)` 需要完整 URL，包含协议头。

**解决**:
```java
// ❌ 错误
new HttpClient("localhost:8080");

// ✅ 正确
new HttpClient("http://localhost:8080");
```

### 如何开启调试日志

```java
httpClient.options().debug(true);
```

### 异步结果如何转为同步

```java
HttpResponse response = Feat.httpClient("https://api.example.com")
        .get("/data")
        .submit()
        .get();  // 阻塞等待结果
```
