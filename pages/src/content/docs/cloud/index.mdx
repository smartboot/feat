本教程将引导你在 5 分钟内创建并运行第一个 Feat Cloud 应用。

## 步骤 1：创建 Maven 项目

使用 IntelliJ IDEA 创建一个新的 Maven 项目，然后在 `pom.xml` 中添加 Feat Cloud 依赖：

```xml title="pom.xml"
<dependency>
    <groupId>tech.smartboot.feat</groupId>
    <artifactId>feat-cloud-starter</artifactId>
    <version>${feat.version}</version>
</dependency>
```

<Aside title="关于版本号">
  `${feat.version}` 需要替换为实际的版本号，建议使用[最新版本](https://central.sonatype.com/artifact/tech.smartboot.feat/feat-cloud-starter)。
</Aside>

## 步骤 2：配置 IntelliJ IDEA（可选）

在较旧版本的 IntelliJ IDEA 中，需要调整配置以确保 Feat Cloud 的静态优化功能正常生效。

**配置步骤：**

1. 打开 IDEA，进入 `Preferences`
2. 导航到 `Build, Execution, Deployment -> Build Tools -> Maven -> Runner`
3. 勾选 `Delegate IDE build/run actions to Maven`
4. 点击 `OK` 保存配置

<img src={idea_setting.src} alt="IDEA 配置截图" width="60%" className="shadow"/>

<Aside type="tip">
  最新版本的 IntelliJ IDEA 已无需此配置，该步骤为可选项。
</Aside>

### 配置 maven-clean-plugin（推荐）

为缓解 IDEA 增量编译可能导致 APT（注解处理器）失效的问题，建议在 `pom.xml` 中配置 `maven-clean-plugin`：

```xml title="pom.xml"
<build>
    <plugins>
        <!-- 自动执行 clean：每次编译前清理 -->
        <plugin>
            <groupId>org.apache.maven.plugins</groupId>
            <artifactId>maven-clean-plugin</artifactId>
            <version>3.3.2</version>
            <!-- 绑定到 validate 阶段（编译前最早执行） -->
            <executions>
                <execution>
                    <id>auto-clean</id>
                    <phase>validate</phase>
                    <goals>
                        <goal>clean</goal>
                    </goals>
                </execution>
            </executions>
        </plugin>
    </plugins>
</build>
```

<Aside type="tip">
  该配置会在每次编译前自动执行 `clean`，确保 APT 生成的代码始终是最新的。
</Aside>

## 步骤 3：编写启动类

在 `src/main/java` 目录下创建 [`Bootstrap.java`](https://gitee.com/smartboot/feat/blob/master/demo/helloworld/src/main/java/tech/smartboot/feat/Bootstrap.java) 文件：

```java title="Bootstrap.java"
@Controller
public class Bootstrap {

    @RequestMapping("/hello")
    public String helloWorld() {
        return "Hello, Feat Cloud!";
    }

    public static void main(String[] args) {
        FeatCloud.cloudServer().listen();
    }
}
```

## 步骤 4：运行应用

点击 IDEA 中的运行按钮，或使用快捷键运行 `Bootstrap` 类。

当看到控制台输出以下信息时，说明应用已成功启动：

```
Feat Router:
 |-> /hello ==> Bootstrap@helloWorld
  ________   ________        _        _________
 |_   __  | |_   __  |      / \      |  _   _  |
   | |_ \_|   | |_ \_|     / _ \     |_/ | | \_|
   |  _|      |  _| _     / ___ \        | |
  _| |_      _| |__/ |  _/ /   \ \_     _| |_
 |_____|    |________| |____| |____|   |_____|
  :: Feat :: (v1.4.0)
🎉Congratulations, the feat startup is successful. cost: 71ms
http://0.0.0.0:8080/
```

这表示服务已在 8080 端口运行，`/hello` 路径已映射到对应的处理方法。

## 步骤 5：验证结果

打开浏览器，访问 `http://localhost:8080/hello`，页面将显示：

```
Hello, Feat Cloud!
```
