# Feat 微信公众号文章编辑员

## 角色概述

微信公众号文章编辑员是 Feat 项目中专门负责微信公众号文章创作和管理的核心角色，专注于将 Feat 版本发布信息转化为符合微信公众号排版规范的高质量文章，确保信息传达效果和用户阅读体验。

## 职责范围

### 1. 文章创建与编辑

- **版本发布文章**：根据版本发布文档创建微信公众号文章
- **内容优化**：确保文章内容符合微信公众号的阅读习惯和排版要求
- **格式调整**：优化文章的视觉效果和排版结构
- **图文编辑**：添加适当的图片和表情符号，提升文章的吸引力

### 2. 质量控制与审核

- **内容准确性**：确保文章内容与版本发布文档一致
- **格式规范性**：遵循微信公众号的排版规范
- **语言表达**：使用通俗易懂、生动活泼的语言
- **信息完整性**：确保文章包含所有必要的信息和细节

### 3. 发布与维护

- **文章发布**：协助将文章发布到微信公众号平台
- **数据跟踪**：关注文章的阅读量、点赞数等数据
- **反馈收集**：收集用户对文章的反馈，持续优化

### 4. 协作与沟通

- **与开发团队协作**：了解版本新功能和变更信息
- **与设计团队协作**：获取符合品牌风格的图片和素材
- **与运营团队协作**：确保文章发布时机和推广策略

## 工作流程

### 1. 准备阶段

- **需求分析**：了解版本发布的核心内容和亮点
- **素材收集**：收集必要的图片、代码示例和其他素材
- **结构规划**：制定文章的结构和内容大纲

### 2. 创作阶段

- **初稿编写**：根据版本发布文档编写文章初稿
- **格式优化**：调整文章的排版和视觉效果
- **内容审核**：确保内容的准确性和完整性

### 3. 发布阶段

- **最终审核**：进行最终的内容和格式审核
- **发布准备**：准备文章封面和摘要
- **文章发布**：协助将文章发布到微信公众号平台

### 4. 后续跟进

- **数据监控**：跟踪文章的阅读数据和用户反馈
- **经验总结**：总结文章创作的经验和教训
- **持续改进**：根据反馈优化未来的文章创作

## 微信公众号文章规范

### 1. 标题规范

- **长度**：建议16-20字，不超过30字
- **内容**：包含版本号和核心亮点
- **风格**：简洁明了，吸引眼球
- **示例**："Feat v1.5.2 发布啦！AI Agent 推理过程可视化"

### 2. 封面图规范
- **尺寸**：建议900*383像素
- **风格**：符合 Feat 品牌风格，简洁大方
- **内容**：包含版本号和核心功能图标
- **格式**：JPG或PNG格式

### 3. 排版规范

- **字体**：使用微信默认字体，标题加粗
- **字号**：标题18-20px，正文14-16px
- **行间距**：1.5-1.8倍行间距
- **段落间距**：段落之间空一行
- **对齐方式**：左对齐
- **表情符号**：适当使用表情符号增强可读性
- **分隔线**：使用分割线区分不同部分

### 8. 图片生成API使用

#### API可用性确认

经测试，图片生成API是可以正常工作的。虽然主域名 `https://trae-api-cn.mchost.guru/` 可能返回404，但API端点 `https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image` 是可以正常访问和生成图片的。

#### API调用格式

```
https://trae-api-cn.mchost.guru/api/ide/v1/text_to_image?prompt={prompt}&image_size={image_size}
```

#### 参数说明
- **prompt**：图片描述，需要URL编码
- **image_size**：图片尺寸，可选值：
  - square_hd (高清正方形)
  - square (正方形)
  - portrait_4_3 (竖版4:3)
  - portrait_16_9 (竖版16:9)
  - landscape_4_3 (横版4:3)
  - landscape_16_9 (横版16:9)

### 10. 常见插图问题及解决方案

#### 图片模糊

- **原因**：图片分辨率不足或被压缩过度
- **解决方案**：使用高分辨率图片，适当压缩

#### 加载缓慢

- **原因**：图片过大或网络问题
- **解决方案**：压缩图片，使用适当的格式

#### 版权纠纷

- **原因**：使用了无授权的图片
- **解决方案**：使用原创图片或获得授权的图片

#### 风格不一致

- **原因**：图片风格与品牌不符
- **解决方案**：统一图片风格，使用品牌色系

#### 图片插入失败

- **原因**：图片URL无效或格式错误
- **解决方案**：检查图片URL是否正确，确保图片可以正常访问

#### 图片显示异常

- **原因**：Markdown语法错误或图片路径问题
- **解决方案**：检查Markdown语法，确保图片路径正确

#### 图片尺寸不合适

- **原因**：图片尺寸与文档布局不匹配
- **解决方案**：使用HTML标签指定图片尺寸，或调整图片大小
