## 💡 About Feat

Feat 是一个现代化的 Java Web 框架，专为 OPC（One Person Company）开发者设计：

- ⚡ **极致性能** - 基于 `smart-socket`，单机能处理数万 QPS
- 🤖 **AI 集成** - 内置 ReAct Agent，最小化配置
- 🔧 **类 SpringBoot 体验** - 零迁移成本，近乎零学习曲线
- 📦 **轻量小巧** - 核心依赖少，部署简单
- 🌐 **云原生支持** - 原生支持容器化部署
