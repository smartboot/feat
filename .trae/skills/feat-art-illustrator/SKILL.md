# Feat 艺术插画生成技能

为微信公众号文章生成手绘风格艺术插画，将抽象概念转化为生动有趣的视觉表达。

## ⚠️ 重要说明

**最终输出格式：PNG/JPG 图片（通过截图生成）**

因此：
- ❌ **不要添加动画效果**（animation、transition 等）
- ❌ **不要添加交互效果**（hover、click 等）
- ✅ **专注于静态视觉效果**
- ✅ **确保截图时的视觉呈现最佳**

## 适用场景

- 微信公众号文章配图
- 技术概念可视化
- 品牌故事讲述
- 产品特性展示
- 情感化内容表达

## 艺术风格特征

### 手绘插画风格

| 特征 | 说明 |
|------|------|
| 线条 | 不规则手绘线条，略带抖动感，模拟真实笔触 |
| 色彩 | 柔和暖色调为主，低饱和度，温暖亲切 |
| 构图 | 留白充足，主体突出，呼吸感强 |
| 图标 | 卡通风格、动物、植物、物品等趣味元素 |
| 阴影 | 柔和渐变，无生硬边界 |

## 工作流程

### 1. 关键词提取

从文章内容中提取核心关键词：

```
输入：文章内容
输出：3-5个核心关键词 + 情感基调
```

### 2. 意象映射

将关键词映射到视觉意象：

| 关键词类型 | 映射规则 |
|------------|----------|
| 技术概念 | 抽象化 + 隐喻图标 |
| 情感表达 | 氛围色 + 情绪图标 |
| 流程步骤 | 路径图 + 方向图标 |
| 对比关系 | 分区布局 + 对比图标 |
| 成长变化 | 渐进构图 + 生长图标 |

### 3. 构图设计

选择合适的构图模板：

- **中心聚焦**：单一主体，周围留白
- **左右对称**：对比展示，平衡美感
- **上下分层**：层次分明，逻辑清晰
- **环绕布局**：核心居中，元素环绕
- **路径引导**：起点终点，过程展示

### 4. 风格渲染

应用手绘风格：

```
1. 绘制不规则边框线条
2. 添加柔和背景色块
3. 放置主题图标元素
4. 绘制连接线条和装饰
5. 添加手写风格文字
6. 点缀装饰性小元素
```

## 技术原理模板设计规范

### 架构图 (art-architecture.html)

**布局特点**：
- 横向分层，每层独立色块
- 左侧标签标注层级名称
- 组件使用圆角矩形
- 连接线使用虚线或箭头

**适用内容**：
- 微服务架构
- 系统模块划分
- 技术栈展示

### 数据流图 (art-dataflow.html)

**布局特点**：
- 横向流程，从左到右
- 节点形状区分类型（圆形=输入/输出，菱形=决策，矩形=处理）
- 数据标签标注在连接线上
- 支持反馈循环展示

**适用内容**：
- 数据处理管道
- ETL 流程
- AI 推理链路

### 时序图 (art-sequence.html)

**布局特点**：
- 参与者横向排列在顶部
- 生命线垂直向下延伸
- 消息用箭头表示方向
- 激活框表示执行期间

**适用内容**：
- API 调用流程
- 消息传递机制
- Hook 拦截过程

### 状态机图 (art-state-machine.html)

**布局特点**：
- 状态用圆形表示
- 初始状态用实心圆点
- 终止状态用双圈
- 转换用带箭头的线表示
- 支持自循环

**适用内容**：
- 请求生命周期
- 订单状态流转
- 工作流引擎

### 网络拓扑图 (art-topology.html)

**布局特点**：
- 节点按功能区域分组
- 不同类型节点使用不同形状
- 连接线区分实线/虚线
- 支持流量指示器动画

**适用内容**：
- 分布式系统架构
- 集群部署方案
- 服务依赖关系

### 分层架构图 (art-layers.html)

**布局特点**：
- 纵向分层，从上到下
- 每层左侧标注层级信息
- 模块横向排列
- 依赖箭头指向下方
- 支持横切关注点展示

**适用内容**：
- 软件分层设计
- DDD 架构
- 洋葱架构

## 输出规范

### 文件命名

```
{文章名}_{插画类型}_{尺寸}_{序号}.html
```

示例：
- `v1.5.2_art-concept_900x500_01.html`
- `v1.5.2_art-process_900x600_01.html`

### 导出要求

1. 格式：PNG 或 JPG
2. 文件大小：≤1MB（文中图）、≤2MB（封面图）
3. 分辨率：72dpi（屏幕显示）
4. 色彩模式：RGB

## 示例场景

### 场景1：技术概念可视化

**输入**："Hook 机制让 AI 推理过程可视化"

**输出**：
- 关键词：Hook、AI、推理、可视化
- 意象：🔍（观察）+ 🧠（思考）+ 🔗（连接）
- 构图：中心展示 AI 大脑，Hook 作为连接桥梁，可视化元素环绕

**输入**："从程序员到独立开发者"

**输出**：
- 关键词：程序员、独立、成长、转变
- 意象：🌱→🌿→🌳（成长）+ 💻→🚀（转变）
- 构图：渐进路径，从左到右展示成长阶段
- 色调：淡绿 + 暖橙

### 场景3：对比展示

**输入**："传统开发 vs 现代开发"

**输出**：
- 关键词：传统、现代、对比、效率
- 意象：🐢 vs 🐇（速度对比）
- 构图：左右对称分区，中间分隔
- 色调：灰蓝 vs 明黄
