# 模板快速索引

> 快速选择合适的模板，配合 SKILL.md 使用。

---

## 一、封面模板选择

```
文章类型 → 模板选择

┌─────────────────┐
│   文章类型？     │
└────────┬────────┘
         │
    ┌────┴────┬─────────────┐
    ↓         ↓             ↓
 技术教程   产品发布       通用
    │         │             │
    ↓         ↓             ↓
tutorial   release      large/medium
```

| 场景 | 模板 | 尺寸 | 特点 |
|------|------|------|------|
| 技术教程封面 | wechat-cover-tutorial.html | 900×383 | 蓝色主调，步骤感 |
| 产品发布封面 | wechat-cover-release.html | 900×383 | 紫色主调，亮点突出 |
| 通用封面大图 | wechat-cover-large.html | 900×383 | 简洁通用 |
| 封面次图 | wechat-cover-medium.html | 900×500 | 第二条及以后 |

## 二、文中插图选择

### 2.1 风格决策

```
内容属性 → 风格选择

┌─────────────────┐
│  内容是什么？    │
└────────┬────────┘
         │
    ┌────┼────────────┐
    ↓    ↓            ↓
 技术类 混合类      情感类
    │    │            │
    ↓    ↓            ↓
 简约  简约+手绘    手绘艺术
```

### 2.2 内容类型 → 模板映射

| 内容类型 | 视觉表达 | 推荐模板 | 风格 |
|---------|---------|---------|------|
| 单一概念 | 图标+关键词 | art-concept.html | 手绘 |
| 对比分析 | 左右分栏+VS | art-compare.html | 手绘 |
| 成长变化 | 时间轴/曲线 | art-growth.html | 手绘 |
| 层次结构 | 堆叠/嵌套 | art-layers.html | 手绘 |
| 情感氛围 | 场景+色调 | art-mood.html | 手绘 |
| 技术流程 | 流程图 | wechat-inline-art.html | 简约 |
| 架构图 | 层次图 | wechat-inline-art.html | 简约 |

### 2.3 模板详情

#### art-concept.html（概念展示）

**适用**：单一概念、核心观点、关键词展示

**内容约束**：
- 主标题：≤12字符
- 副标题：≤20字符
- 关键词：≤3个
- 图标：1个主图标 + 4个环绕图标

**示例场景**：
- "AI 推理可视化"
- "云原生架构"
- "高性能设计"

---

#### art-compare.html（对比展示）

**适用**：两个概念对比、方案选择、优劣势分析

**内容约束**：
- 对比标题：≤10字符
- 每侧特点：≤3条
- 每条文字：≤10字符

**示例场景**：
- "Feat vs Spring"
- "同步 vs 异步"
- "传统 vs 云原生"

---

#### art-growth.html（成长展示）

**适用**：成长过程、变化历程、阶段演进

**内容约束**：
- 阶段数量：≤4个
- 每阶段标题：≤8字符
- 每阶段描述：≤15字符

**示例场景**：
- "从0到1的架构演进"
- "技术成长之路"
- "产品迭代历程"

---

#### art-layers.html（分层展示）

**适用**：层次结构、架构分层、系统组成

**内容约束**：
- 层级数量：≤4层
- 每层标题：≤10字符
- 每层描述：≤20字符

**示例场景**：
- "系统架构分层"
- "技术栈组成"
- "数据流向"

---

#### art-mood.html（氛围展示）

**适用**：情感表达、场景营造、品牌调性

**内容约束**：
- 主文字：≤15字符
- 辅助文字：≤30字符
- 装饰元素：适量

**示例场景**：
- "新年祝福"
- "团队文化"
- "产品愿景"

---

#### wechat-inline-art.html（通用插图）

**适用**：技术流程、架构图、数据展示

**内容约束**：
- 标题：≤15字符
- 内容项：≤4个
- 每项文字：≤10字符

**示例场景**：
- "请求处理流程"
- "模块依赖关系"
- "性能优化步骤"

## 三、快速决策流程

```
1. 确定图片位置
   ├── 封面 → 选择封面模板
   └── 正文 → 进入步骤2

2. 分析内容类型
   ├── 单一概念 → art-concept.html
   ├── 对比分析 → art-compare.html
   ├── 成长变化 → art-growth.html
   ├── 层次结构 → art-layers.html
   ├── 情感氛围 → art-mood.html
   └── 技术流程 → wechat-inline-art.html

3. 检查内容约束
   └── 确保不超出安全上限

4. 填充内容
   └── 遵循模板结构
```
